### 1.3 GUARDRAILS (mandatory — read before touching any template)

This is the channel most likely to hurt you if mishandled. Plain-English version of why:

- **Google can see the network.** Same owner usually means shared registrant/WHOIS history, shared hosting or Cloudflare account, shared GA4/GTM IDs, shared favicon/CMS fingerprints, similar templates. SpamBrain clusters sites on exactly these signals. Once clustered, *reciprocal or systematic* interlinking between cluster members gets devalued — and in gambling, devaluation can spill into distrust of the whole profile. You cannot hide ownership; you can only make the links so obviously editorial that they'd survive a human review.
- **The test for every network link:** "If a Google webspam analyst with all three sites open side-by-side read this paragraph, would the link look like something an independent editor would have added?" If no — don't place it.

Hard rules:

1. **Ceiling: 1–2 contextual links per network site, total.** Not per page, not per month — *total live at any time*. With 787 referring domains, 2 network domains is ~0.25% of the profile. That's the safe zone. 10+ interlinks between three same-owner gambling sites is a pattern, not a profile.
2. **In-body contextual only.** No sitewide links, no footer links, no blogroll, no "partner sites" widget, no author-bio links. Sitewide same-owner links are the #1 trigger for link-network classification.
3. **No reciprocal circles.** If geekygambler → OCL and onlinecasinoperu → OCL, then **OCL must not link back** to either (beyond an unavoidable mention, which should be nofollow). A→B, C→B is a hub pattern; A↔B↔C↔A is a scheme.
4. **Vary anchors; zero exact-match money anchors between owned sites.** Branded, topical, or descriptive phrases only (see Section 4).
5. **Stagger placement.** Add G1/G2 this week (World Cup timing justifies it), P1/P2 three to four weeks later. Don't push 4 same-owner links live on the same day.
6. **Link inside existing, indexed, trafficked content** — not in new thin posts created to host the link.
7. **Never scale this channel.** Its entire value is 2–4 free, perfectly relevant links. The marginal 5th link adds ~0 equity and meaningful classification risk. When tempted to add more, spend that effort on Channel 2 instead.
8. **Disclosure hygiene:** all three sites should have consistent, honest "About"/ownership pages. Hiding common ownership while interlinking is worse than being open about it.

### 3.2 Red flags — walk away immediately

- **"Casino link packages"** (e.g., "50 DA50+ casino backlinks for $499"). These are PBNs. Every list-seller's inventory is also being sold to 500 other casinos — shared footprint, shared fate.
- Sites with **DR/DA but no organic traffic** (metric inflation via expired domains or link spam). Traffic is the un-fakeable metric.
- **Outbound-link sewers:** existing posts linking to casino + CBD + loans + essay-writing in the same sidebar/articles. Classic farm signature.
- "**Write for us casino**" pages — publicly advertising paid casino links means Google's crawlers see it too.
- Dropped/repurposed domains (check Wayback: was this a Polish dentist's site in 2022?).
- Sites that **won't show you the URL before payment**, sell homepage links, or guarantee "permanent dofollow" in writing on a public marketplace.
- **Language/geo mismatch at scale:** 50 Hindi or Indonesian blog links to a Spanish LATAM site is a pattern screaming "bought."
- Anyone promising specific DR increases or rankings.

### 3.3 Vetting checklist (every placement, no exceptions)

- [ ] **Organic traffic > 1,000/mo** (Ahrefs/Semrush), with a stable or rising 12-month trend — no cliff patterns
- [ ] Traffic comes from **target geos/language** (ES traffic from MX/CO/AR/PE/CL preferred; EN acceptable for trade press)
- [ ] **Real editorial content** published independently of paid posts (check the homepage and category feeds: would a human read this site?)
- [ ] **Outbound-link profile clean:** < ~25% of recent posts carrying obvious paid links; no toxic-niche clusters
- [ ] Ranks for its own brand name + has non-zero branded search
- [ ] Topical adjacency: sports, finance/fintech, entertainment, tech, regional news — not "general blog about everything"
- [ ] Indexed in Google (site: check) and the specific page section you'd appear in gets indexed
- [ ] Domain history clean in Wayback (no niche flips)
- [ ] Your article will be **unique, useful, 800+ words, in correct Spanish (or native English)** — you supply the content, never the seller's $10 filler
- [ ] Link is contextual, in-body, to a *country hub or guide page* (rotate targets; don't pile everything on the homepage)
- [ ] Price sanity: if a "DR 65, 30k traffic" site charges $80, it's fake; if a no-traffic site charges $400, it's a rip — both fail

**Process tip:** keep a vetting spreadsheet (domain, DR, traffic, geo %, price, target URL, anchor, live date, status). Re-check placements quarterly — sellers delete or nofollow links; budget ~10% for replacements.

### 3.4 Cadence & expectations

- **Months 1–3:** 5–6 placements/mo (mid-tier Spanish-language heavy) while PR launches. Slow start looks natural after any prior flat period.
- **Months 4–9:** 8–10/mo blended (add 1–2 premium/trade per month).
- **Months 10–12:** 8–12/mo, rebalancing toward whatever the PR channel isn't covering (often Chile/Ecuador-relevant geo links).
- **Velocity guardrail:** never more than ~15 new paid placements in any month at this DR, and never let paid placements exceed ~60% of new referring domains in a quarter — PR + natural pickup must visibly dilute the bought layer.

### 3.5 Honest risk note on paid links

Buying links that pass PageRank violates Google's spam policies, full stop. The mitigation is not secrecy tricks — it's *quality indistinguishability*: real sites, real traffic, real content, sane anchors, sane velocity, diluted by genuinely earned links. Budget assuming ~20–30% of paid links lose value within a year (deleted, nofollowed, devalued). That depreciation is priced into the cadence above. Keep a disavow file ready but use it only for genuinely toxic acquired/negative-SEO patterns — pre-emptive mass disavow does more harm than good.
